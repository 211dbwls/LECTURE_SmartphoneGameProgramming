중간고사 리뷰
- 자바, 코틀린을 사용하면 둘 다 같이 사용 가능
- Package Name은 구입하지 않아도 사용 가능
- Manifest : 파일 설정 build.gradle : 빌드 환경
- 실기기 개발용으로 전환
- String Resource 그냥 ""으로 쓰는 게 속도가  더 빠름. 다국어 지원을 위해 바꾸는 것
- Resource Filename : 대문자 사용 불가
- thread safe 사용 x
- padding(background o), margin(background x)
- garbage collection  
- view tag : int외 다른 정보도 사용 가능(아이폰은 int만 저장 가능)
- save, restore
- game loop : while 사용 x
- frametime 단위 : nanos
- px : 권장되지 않음
- dp : 넓이에 비례
- 단위 없이 사용할 수 없음
- builder를 사용하지 않을 시, 객체를 완전하지 않은 상태에서 생성하거나, 생성자에 모든 걸 넣어야 함
